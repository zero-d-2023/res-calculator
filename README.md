# 🎬 Resolution Master

**Resolution Master**는 영상 컨텐츠 제작자를 위한 전문적이고 직관적인 해상도 및 화면비 계산기입니다. 4K, QHD, FHD 등 표준 규격을 기반으로 자유롭게 해상도를 조절하고, 시각적으로 비교할 수 있습니다.

## ✨ 주요 기능

- **듀얼 해상도 입력 시스템**: 원래 해상도(Base)와 조절된 해상도(Scaled)를 한눈에 비교하고 동시에 관리합니다.
- **정밀 화면비 유지**: 자물쇠 기능을 통해 어떤 해상도에서도 완벽한 화면비를 유지하며 리사이징할 수 있습니다.
- **스마트 짝수 보정(Even Snap)**: 영상 인코딩 최적화를 위해 가로/세로 해상도를 즉시 가장 가까운 짝수 정수로 보정합니다.
- **스케일 가이드**: 원본 대비 현재 해상도의 크기를 실시간 그래픽으로 비교하여 보여줍니다.
- **1픽셀 오차 감지**: 반올림으로 발생하는 미세한 오차를 감지하고, 원본 비율과의 관계를 상세히 설명해 줍니다.

## 🎨 디자인 특징

- **Glassmorphism**: 투명하고 세련된 유리 질감의 UI 디자인.
- **Dark Mode**: 장시간 작업에도 눈이 편안한 깊이 있는 다크 테마.
- **Premium Aesthetics**: Outfit 폰트와 부드러운 애니메이션이 적용된 프리미엄 사용자 경험.

## 🚀 시작하기

이 프로젝트는 별도의 설치 없이 브라우저에서 바로 실행 가능한 정적 웹 페이지입니다.

1. `index.html` 파일을 브라우저로 엽니다.
2. 원하는 원본 해상도를 선택하거나 직접 입력합니다.
3. 퍼센티지 버튼이나 직접 입력을 통해 해상도를 조절합니다.
